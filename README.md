# Aplicação para Estaciomento - Cadastros e Exclusão de Veículos

## Instalar Node JS
```
npm install node
```

## Instalar Vue JS
```
npm install vue
```

### Rodar o Projeto
```
npm run serve
```

### Rodar o JSON Server
```
npm run backend
```

### Compiles and minifies for production
```
npm run build
```
